CSS Selectors And Specificity

Selector Type Degree Of Specificity And Its Use Visual Example
Type This is the lowest degree of specificity and is used to make styles easy to edit h1 {
color: red;
}
Class More specific than a type selector and can be reused over many elements .headline {
color: blue;
}
ID This is the highest degree of specificity. It overrides the styles of both type and class selectors. It is meant to be used sparingly for example, on elements that must always look the same. #large-title {
color: green;
}
